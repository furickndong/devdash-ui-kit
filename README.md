# DevDash UI Kit

## 📌 Description
Dashboard admin construit avec un design system basé sur les variables CSS.

## 🎯 Objectifs
- Utilisation des CSS variables
- Design cohérent et scalable
- Responsive design
- Dark mode

## 🏗️ Structure
- variables.css → variables globales
- base.css → reset + base
- layout.css → structure
- components.css → composants UI

## ✨ Fonctionnalités
- Thème dark
- Hover & transitions
- Sidebar active
- Responsive mobile
- Variables locales dans .card
- Thème alternatif

## 🚀 Lancer le projet
Ouvrir index.html dans un navigateur
